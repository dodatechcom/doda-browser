Privacy Policy

Last updated: May 2026

Data Collection

DurgaShield does not collect, store, or transmit any personal data, browsing history, or identifiable information to any external server.

Local Storage

The following data is stored locally in your browser using chrome.storage.local and never leaves your device:

  • Extension configuration (enabled/disabled features)
  • Blocking statistics (total blocked count, daily breakdown)
  • Site-specific whitelist/blacklist entries
  • Filter list update timestamps
  • Custom user-defined rules
  • Disabled rule IDs

Network Requests

The extension makes no network requests of its own. All filtering rules are bundled with the extension or fetched from known filter list sources (e.g., uBlock Origin's filter lists) only when you manually trigger an update. No browsing data is sent anywhere.

Declarative Net Request

DurgaShield uses Firefox's declarativeNetRequest API to block requests locally. All filtering happens on your device. No data about blocked or allowed requests is transmitted externally.

Third-Party Services

This extension does not use any third-party analytics, tracking, or telemetry services.

Changes

If this privacy policy changes, the updated version will be reflected here. Since no data is collected, there is nothing to notify you about.

Contact

For questions about this privacy policy, please open an issue on the extension's repository.
