About Doda Intelligence Hub

Doda Intelligence Hub is the command center for all Doda Browser's unique capabilities. It unifies four powerful services into one interface, with a global command palette accessible from any page.

  • Knowledge Graph — Full-text search across every page you've visited, automatically indexed
  • AI Assistant — Private, local AI powered by Ollama — chat, analyze tab content, compare pages, and debate topics
  • Identity Wallet — Passwordless login using Ed25519 cryptographic keys stored securely in your browser
  • Provenance (Trust OS) — Extract factual claims from pages, compare sources, and track how information spreads

Quick Tips:

  • Press Ctrl+K or Ctrl+Shift+, anywhere in the browser to open the Command Palette
  • Visit individual service pages: about:knowledgegraph, about:ai, about:identity, about:provenance
  • Make sure Ollama is running (ollama serve) for AI & Provenance features
